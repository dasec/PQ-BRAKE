◆ compute_a_x()

NTL::ZZ_pE Client::compute_a_x ( )

Computes a_x value in modified OPRF protocol based on the secret polynomial stored in the Client object. Takes the coefficients of the secret polynomial, concatenates them - thereby creating h, then new coefficients (a0...aN) are created by hashing "0h","1h","2h"..."nh", lastly converting the hashes to integers and performing a 'mod q' operation.
